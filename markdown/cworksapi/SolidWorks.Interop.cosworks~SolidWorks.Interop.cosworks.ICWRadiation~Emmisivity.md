# Emmisivity Property (ICWRadiation)

Gets or sets the value for emissivity for this radiation.

## Syntax

### Visual Basic (Declaration)

```vb
Property Emmisivity As System.Double
```

### Visual Basic (Usage)

```vb
Dim instance As ICWRadiation
Dim value As System.Double

instance.Emmisivity = value

value = instance.Emmisivity
```

### C#

```csharp
System.double Emmisivity {get; set;}
```

### C++/CLI

```cpp
property System.double Emmisivity {
   System.double get();
   void set (    System.double value);
}
```

### Property Value

Emissivity
