# Flags Property (IEdmSWBomColumn)

Gets or sets the alignment of this BOM column.

## Syntax

### Visual Basic

```vb
Property Flags As System.Integer
```

### C#

```csharp
System.int Flags {get; set;}
```

### C++/CLI

```cpp
property System.int Flags {
   System.int get();
   void set (    System.int value);
}
```

### Property Value

0 = left

1 = right

2 = center
