# Show Most Recent Measurement Results Example (JavaScript)

## eDrawings
API Help

This example shows how to display a webpage message box containing the most
recent measurement results.

1. Ensure that

  C:\block.sldprt

  exists.
2. C

  lick

  Load File

  .

  The

  block.sldprt

  document is displayed in the eDrawing

  s

  Control in the Help topic.
3. Click

  Measure

  and click an edge on

  block.sldprt

  .

  The measurements of the edge are displayed

  in the
  eDrawings Control in

  the Help topic

  .
4. Click

  Measure Result

  .

  The most recent measurement results, i.e.,

  those

  of the edge clicked in step 3,

  are displayed in a webpage message box.
5. Click

  OK

  to close the message box.
