# IPDMWProject Interface Properties

For a list of all members of this type, see[IPDMWProject members](PDMWorks.Interop.pdmworks~PDMWorks.Interop.pdmworks.IPDMWProject_members.html).

## Public Properties

|  | Name | Description |
| --- | --- | --- |
| Property | AllowRead | Gets whether the user has read access to this project. |
| Property | AllowWrite | Gets whether the user has write access to this project. |
| Property | Description | Gets or sets the description for this project. |
| Property | DocumentCount | Gets the number of documents in this project. |
| Property | DocumentList | Gets a the list of document names for this project. |
| Property | Documents | Gets the documents for this project. |
| Property | Name | Gets or sets the name of this project. |
| Property | Parent | Gets or sets the parent project. |
| Property | SubProjects | Gets the child projects for this project. |
