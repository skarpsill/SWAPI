# BlockInstance

## BlockInstance Object

This object is obsolete and has been superseded
by[SketchBlockInstance](sldworksAPI.chm::/SketchBlockInstance/SketchBlockInstance.htm).

The BlockInstance interface allows access to blocks. Using this interface,
you can:

- Traverse through all of the block instances associated
  with a given view or sheet
- Get the Annotation interface, which gives you
  access to various APIs (for example, select, leader information, and so
  on)
- Get the BlockDefinition interface
- Get or set the block instance's:
- Get all attributes and the number of attributes
  on this block instance

Use theAccessorslink to get a list of functions that return this object.
