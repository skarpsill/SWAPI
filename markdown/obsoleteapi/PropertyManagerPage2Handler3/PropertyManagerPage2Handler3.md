# PropertyManagerPage2Handler3

## PropertyManagerPage2Handler3 Object

This object is obsolete and has been superseded
by[PropertyManagerPage2Handler4](../PropertyManagerPage2Handler4/PropertyManagerPage2Handler4.htm).

Must be implemented by the add-in application to
handle callbacks from PropertyManagerPage2.

NOTE:To
use this interface in a VBA macro, you must reference theSolidWorks
exposed type libraries for add-in usetype library specific to
the version of SolidWorks that you are running.

For all unimplemented methods of this object, the
add-in application can return E_NOTIMPL.
