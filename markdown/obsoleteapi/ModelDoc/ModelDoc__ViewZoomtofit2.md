# ModelDoc::ViewZoomtofit2

This method is obsolete
and has been superseded by[ModelDoc2::ViewZoomtofit2](sldworksAPI.chm::/ModelDoc2/ModelDoc2__ViewZoomtofit2.htm).

Description

This method zooms the currently active view
to fit the screen.

Syntax (OLE Automation)

void ModelDoc.ViewZoomtofit2 ( )

Syntax (COM)

status = ModelDoc->ViewZoomtofit2 ( )

(Table)=========================================================

| Return: | (HRESULT) status | S_OK if successful |
| --- | --- | --- |

Remarks

This method differs from ModelDoc::ViewZoomtofit
in that it zooms in closer to the geometry and has less of a border.
