# ModelDoc::WindowRedraw

This method is obsolete
and has been superseded by[ModelDoc2::WindowRedraw](sldworksAPI.chm::/ModelDoc2/ModelDoc2__WindowRedraw.htm).

Description

This method redraws the current window. The current window includes
the FeatureManager design tree view and the graphics view or views if
you have used the window splitter.

Syntax (OLE Automation)

void ModelDoc.WindowRedraw ()

Syntax (COM)

status = ModelDoc->WindowRedraw
( )

(Table)=========================================================

| Return: | (HRESULT) status | S_OK if successful |
