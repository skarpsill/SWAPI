# ModelDoc::InsertRoutePoint

This
method is obsolete and has been superseded by[ModelDoc2::InsertRoutePoint](sldworksAPI.chm::/ModelDoc2/ModelDoc2__InsertRoutePoint.htm).

Description

This method addds a route point based on the
selected point. If the selection set is not complete, then theInsert
Route Pointdialog is displayed.

Syntax (OLE Automation)

void ModelDoc.InsertRoutePoint ( )

Syntax (COM)

status = ModelDoc->InsertRoutePoint ( )

(Table)=========================================================

| Return: | (HRESULT) status | S_OK if successful |
| --- | --- | --- |

Remarks

The route point is the point on the fitting that
aligns with the sketch point when the fitting is inserted.
