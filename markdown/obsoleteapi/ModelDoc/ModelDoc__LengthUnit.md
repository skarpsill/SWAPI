# ModelDoc::LengthUnit

This
property is obsolete and has been superseded by[ModelDoc2::LengthUnit](sldworksAPI.chm::/ModelDoc2/ModelDoc2__LengthUnit.htm).

and set the same LengthUnit value used by ModelDoc::GetUnits and ModelDoc::SetUnits.
