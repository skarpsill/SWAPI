# DrawingDoc::GetEditBlock

This method is obsolete and has been superseded
by[SketchManager::EditSketchBlock](sldworksAPI.chm::/SketchManager/SketchManager__EditSketchBlock.htm)and[SketchManager::EndEditSketchBlock](sldworksAPI.chm::/SketchManager/SketchManager__EndEditSketchBlock.htm).

Description

This method gets whether or
not a block is being edited in the drawing.

Syntax (OLE Automation)

retval = DrawingDoc.GetEditBlock ()

(Table)=========================================================

| Output: | (VARIANT_BOOL) retval | TRUE if a block is being edited,
FALSE if not |
| --- | --- | --- |
