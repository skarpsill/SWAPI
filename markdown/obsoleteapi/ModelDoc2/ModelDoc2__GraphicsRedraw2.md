# ModelDoc2::GraphicsRedraw2

This method is obsolete and has been superseded
by[ModelView::GraphicsRedraw](sldworksAPI.chm::/ModelView/ModelView__GraphicsRedraw.htm).

Description

This method forces a repaint of the current graphics view.

Syntax (OLE Automation)

ModelDoc2.GraphicsRedraw2 ()

Syntax (COM)

status = ModelDoc2->GraphicsRedraw2
( )

(Table)=========================================================

| Return: | (HRESULT) status | S_OK if successful |
| --- | --- | --- |

Remarks

This method forces the display to be updated immediately
when called.
