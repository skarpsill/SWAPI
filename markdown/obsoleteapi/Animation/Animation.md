# Animation

## Animation Object

This interface is:

- obsolete and has not been
  superseded.
- nonfunctional in SolidWorks
  2008 and later.

  Use the interfaces related to motion studies introduced in SolidWorks
  2008 to access animation and simulation.

(Table)=========================================================

| image\Feature.gif |  |  |
| --- | --- | --- |
|  | image\branch1.gif |  |
|  |  | image\branch1.gif |

Allows access to an animation and theAnimation
Controllerpop-up toolbar, which is currently only available when
replaying a Physical Simulation in the SolidWorks API.

NOTE:Do not confuse theAnimation Controllerpop-up toolbar
available in the core SolidWorks software with SolidWorks Animator, a
SolidWorks add-in. SolidWorks Animator provides more functionality than
theAnimation Controllerpop-up
toolbar. See the SolidWorks Animator online Help for details.

Use theAccessorslink to obtain a list of functions that return this object.
