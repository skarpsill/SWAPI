# Animation::PlayMode

This property
is:

- obsolete and has not been
  superseded.
- nonfunctional in SolidWorks
  2008 and later.

  Use the interfaces related to motion studies introduced in SolidWorks
  2008 to access animation and simulation.

Description

This method gets mode
in which this animation is playing.

Syntax (OLE Automation)

VB_GET

Mode = Animation.PlayMode (VB Get property)

END_VB_GET

VB_SET

Animation.PlayMode = Mode (VB Set
property)

END_VB_SET

Mode = Animation.GetPlayMode ( ) (C++ Get property)

END_VC_GET

VC_SET

Animation.SetPlayMode ( Mode ) (C++
Set property)

END_VC_SET

(Table)=========================================================

| Output: | (long) Mode | Mode in which this animation
is playing as defined in swAnimationPlayMode_e |
| --- | --- | --- |

Syntax (COM)

COM_GET

status = Animation->get_PlayMode
( &Mode )

END_COM_GET

COM_SET

status = Animation->put_PlayMode
( Mode )

END_COM_SET

Parameters Table Start

(Table)=========================================================

is playing as defined in swAnimationPlayMode_e |
| --- | --- | --- |
| Return: | (HRESULT) status | S_OK if successful |
