# View::SetDisplayMode

This method is obsolete and has been superseded
by[View::SetDisplayMode2](View__SetDisplayMode2.htm).

Description

This method will set the display mode of the drawing view.

Syntax (OLE Automation)

void View.SetDisplayMode ( displayIn)

(Table)=========================================================

| Input: | (long) displayIn | Display mode of the drawing view as defined in swDisplayMode_e |
| --- | --- | --- |

Syntax (COM)

status = View->SetDisplayMode (
displayIn )

(Table)=========================================================

| Input: | (long) displayIn | Display mode of the drawing view as defined in swDisplayMode_e |
| --- | --- | --- |
| Return: | (HRESULT)status | S_OK if successful |
