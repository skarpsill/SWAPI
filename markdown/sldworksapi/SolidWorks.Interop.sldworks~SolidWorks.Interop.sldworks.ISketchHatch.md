# ISketchHatch Interface

Represents an area hatch, which is inserted into a SOLIDWORKS drawing polygon or component face when you click

Insert > Annotations > Area Hatch/Fill

in a SOLIDWORKS drawing.

**NOTE:**Click the**Members**link, located near the top of the topic, to see this interface's methods and properties.
