# PropertyManagerHandleModuleName Property (IMacroFeatureData)

Gets or sets the name of the module in the macro file from or to the PropertyManager handle.

## Syntax

### Visual Basic (Declaration)

```vb
Property PropertyManagerHandleModuleName As System.String
```

### Visual Basic (Usage)

```vb
Dim instance As IMacroFeatureData
Dim value As System.String

instance.PropertyManagerHandleModuleName = value

value = instance.PropertyManagerHandleModuleName
```

### C#

```csharp
System.string PropertyManagerHandleModuleName {get; set;}
```

### C++/CLI

```cpp
property System.String^ PropertyManagerHandleModuleName {
   System.String^ get();
   void set (    System.String^ value);
}
```

NOTE:

See

[Differences Between Unmanaged C++ and C++/CLI Code](DifferencesBetweenUnManagedAndCPPCLI.htm)

.

### Property Value

Name of the module in the macro file
