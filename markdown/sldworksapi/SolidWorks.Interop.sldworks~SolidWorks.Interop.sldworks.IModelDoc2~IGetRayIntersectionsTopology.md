# IGetRayIntersectionsTopology Method (IModelDoc2)

Gets the topology intersections generated by

[IModelDoc2::IRayIntersections](SOLIDWORKS.Interop.sldworks~SOLIDWORKS.Interop.sldworks.IModelDoc2~IRayIntersections.html)

.

### Return Value

- in-process, unmanaged C++: Pointer to an array containing the hit entities (see

  Remarks

  )

VBA, VB.NET, C#, and C++/CLI: Not supported

See[In-process Methods](sldworksapiprogguide.chm::/OVERVIEW/In-process_Methods.htm)for details about this type of method.

## Remarks

This method only returns information if the swRayPtsOptsTOPOLS option was specified in the Options argument of[IModelDoc2::RayIntersections](SOLIDWORKS.Interop.sldworks~SOLIDWORKS.Interop.sldworks.IModelDoc2~RayIntersections.html).

The array returned contains information about each entity hit of each ray on the given bodies. The topology hit can be coordinated with the hit point using the BodyIndex return value from[IModelDoc2::IGetRayIntersectionsPoints](SOLIDWORKS.Interop.sldworks~SOLIDWORKS.Interop.sldworks.IModelDoc2~IGetRayIntersectionsPoints.html). You can determine the type of item hit (face, edge or vertex) by checking the IntersectionType return value in the IModelDoc2::IGetRayIntersectionsPoints.
