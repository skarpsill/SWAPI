# GetTextAtIndex Method (INote)

Gets the specified text in this note.

## Syntax

### Visual Basic (Declaration)

```vb
Function GetTextAtIndex( _
   ByVal Index As System.Integer _
) As System.String
```

### Visual Basic (Usage)

```vb
Dim instance As INote
Dim Index As System.Integer
Dim value As System.String

value = instance.GetTextAtIndex(Index)
```

### C#

```csharp
System.string GetTextAtIndex(
   System.int Index
)
```

### C++/CLI

```cpp
System.String^ GetTextAtIndex(
   System.int Index
)
```

NOTE:

See

[Differences Between Unmanaged C++ and C++/CLI Code](DifferencesBetweenUnManagedAndCPPCLI.htm)

.

### Parameters

- `Index`: Index position of the text; index begins at 1

### Return Value

Text string
