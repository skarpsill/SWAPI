# AlignHorz Method (IDrawingDoc)

Uses the selected edge to align the current drawing view.

## Syntax

### Visual Basic (Declaration)

```vb

```

```vb

instance.AlignHorz()
```

```

```

[DrawingDoc::AlignHorz](ms-its:sldworksapivb6.chm::/sldworks~DrawingDoc~AlignHorz.html)

## Remarks

This method aligns the drawing view so that the selected edge is horizontal.
