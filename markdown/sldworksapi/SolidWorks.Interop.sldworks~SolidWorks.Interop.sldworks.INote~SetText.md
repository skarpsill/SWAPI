# SetText Method (INote)

Sets the text of the note.

## Syntax

### Visual Basic (Declaration)

```vb
Function SetText( _
   ByVal Txt As System.String _
) As System.Boolean
```

### Visual Basic (Usage)

```vb
Dim instance As INote
Dim Txt As System.String
Dim value As System.Boolean

value = instance.SetText(Txt)
```

### C#

```csharp
System.bool SetText(
   System.string Txt
)
```

### C++/CLI

```cpp
System.bool SetText(
   System.String^ Txt
)
```

NOTE:

See

[Differences Between Unmanaged C++ and C++/CLI Code](DifferencesBetweenUnManagedAndCPPCLI.htm)

.

### Parameters

- `Txt`: Text to replace the current note text

### Return Value

True if the text is set, false if not
