# IDrawingComponent Interface Properties

For a list of all members of this type, see[IDrawingComponent members](SolidWorks.Interop.sldworks~SolidWorks.Interop.sldworks.IDrawingComponent_members.html).

## Public Properties

|  | Name | Description |
| --- | --- | --- |
| Property | Component | Gets the assembly component referenced in this drawing component. |
| Property | Layer | Gets or sets the name of the layer on which the component resides in the view. |
| Property | LayerOverride | Gets or sets whether the drawing component has properties that override the default properties of the layer. |
| Property | Name | Gets the name of the drawing component. |
| Property | Style | Gets or sets the style for the line for this component in this drawing view. |
| Property | UseDocumentDefaults | Gets or sets whether to use the document default settings for the component's line fonts. |
| Property | View | Gets the drawing view on which this component resides. |
| Property | Visible | Gets or sets the visibility state for this component for this drawing view. |
| Property | Width | Gets or sets the width of the line for this component for this drawing view. |
