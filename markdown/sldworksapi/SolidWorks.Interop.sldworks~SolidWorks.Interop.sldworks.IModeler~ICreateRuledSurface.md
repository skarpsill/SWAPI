# ICreateRuledSurface Method (IModeler)

Creates a ruled surface from the curves.

## Syntax

### Visual Basic (Declaration)

```vb
Function ICreateRuledSurface( _
   ByVal Curve1In As Curve, _
   ByVal Curve2In As Curve, _
   ByRef Apex As System.Double _
) As Surface
```

### Visual Basic (Usage)

```vb
Dim instance As IModeler
Dim Curve1In As Curve
Dim Curve2In As Curve
Dim Apex As System.Double
Dim value As Surface

value = instance.ICreateRuledSurface(Curve1In, Curve2In, Apex)
```

### C#

```csharp
Surface ICreateRuledSurface(
   Curve Curve1In,
   Curve Curve2In,
   ref System.double Apex
)
```

### C++/CLI

```cpp
Surface^ ICreateRuledSurface(
   Curve^ Curve1In,
   Curve^ Curve2In,
   System.double% Apex
)
```

NOTE:

See

[Differences Between Unmanaged C++ and C++/CLI Code](DifferencesBetweenUnManagedAndCPPCLI.htm)

.

### Parameters

- `Curve1In`: First

[curve](SOLIDWORKS.Interop.sldworks~SOLIDWORKS.Interop.sldworks.ICurve.html)
- `Curve2In`: Second

[curve](SOLIDWORKS.Interop.sldworks~SOLIDWORKS.Interop.sldworks.ICurve.html)
- `Apex`: Array of 3 doubles, (x, y, z), the apex point

### Return Value

Newly created ruled

[surface](SOLIDWORKS.Interop.sldworks~SOLIDWORKS.Interop.sldworks.ISurface.html)

[Modeler::ICreateRuledSurface](ms-its:sldworksapivb6.chm::/sldworks~Modeler~ICreateRuledSurface.html)

.

## Remarks

Valid input curves are b-curves.
