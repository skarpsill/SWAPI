# Reset Method (ISplineHandle)

Resets the selected handle to its initial state on this spline handle.

## Syntax

### Visual Basic (Declaration)

```vb
Sub Reset()
```

### Visual Basic (Usage)

```vb
Dim instance As ISplineHandle

instance.Reset()
```

### C#

```csharp
void Reset()
```

### C++/CLI

```cpp
void Reset();
```
