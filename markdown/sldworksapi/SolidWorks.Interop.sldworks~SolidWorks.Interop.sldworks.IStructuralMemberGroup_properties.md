# IStructuralMemberGroup Interface Properties

For a list of all members of this type, see[IStructuralMemberGroup members](SolidWorks.Interop.sldworks~SolidWorks.Interop.sldworks.IStructuralMemberGroup_members.html).

## Public Properties

|  | Name | Description |
| --- | --- | --- |
| Property | AlignAxis | Gets and sets the axis on which to align this structural-member group. |
| Property | AlignmentVector | Gets and sets a reference vector for this structural-member group. |
| Property | Angle | Gets and sets the rotational angle of this structural-member group. |
| Property | ApplyCornerTreatment | Gets and sets whether or not to apply a corner treatment to this structural-member group. |
| Property | CornerTreatmentType | Gets and sets the type of corner treatment for this structural-member group. |
| Property | GapForOtherGroups | Gets and sets the gap between segments in different structural-member groups. |
| Property | GapWithinGroup | Gets and sets the gap between connected segments within this structural-member group. |
| Property | LocateProfilePoint | Gets and sets the point to use to locate the profile for this structural-member group. |
| Property | MergeArcSegmentBodies | Gets or sets whether to merge arc segment bodies with adjacent bodies in this structural-member group. |
| Property | MirrorProfile | Gets and sets whether to mirror the profile of this structural-member group. |
| Property | MirrorProfileAxis | Gets and sets the axis about which to mirror the profile of this structural-member group. |
| Property | MiterMergeCondition | Get or set whether to merge miter trimmed bodies in this structural-member group. |
| Property | Segments | Gets and sets the sketch segments to use in this structural-member group. |
