# ReverseEndTangentDirection Property (ILoftFeatureData)

Reverses the tangent direction for the end point of this loft feature.

## Syntax

### Visual Basic (Declaration)

```vb
Property ReverseEndTangentDirection As System.Boolean
```

### Visual Basic (Usage)

```vb
Dim instance As ILoftFeatureData
Dim value As System.Boolean

instance.ReverseEndTangentDirection = value

value = instance.ReverseEndTangentDirection
```

### C#

```csharp
System.bool ReverseEndTangentDirection {get; set;}
```

### C++/CLI

```cpp
property System.bool ReverseEndTangentDirection {
   System.bool get();
   void set (    System.bool value);
}
```

NOTE:

See

[Differences Between Unmanaged C++ and C++/CLI Code](DifferencesBetweenUnManagedAndCPPCLI.htm)

.

### Property Value

True reverses the end tangent direction, false does not
