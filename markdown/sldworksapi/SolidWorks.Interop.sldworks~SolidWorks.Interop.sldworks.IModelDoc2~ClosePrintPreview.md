# ClosePrintPreview Method (IModelDoc2)

Closes the currently displayed Print Preview page for this document.

## Syntax

### Visual Basic (Declaration)

```vb
Sub ClosePrintPreview()
```

### Visual Basic (Usage)

```vb
Dim instance As IModelDoc2

instance.ClosePrintPreview()
```

### C#

```csharp
void ClosePrintPreview()
```

## Remarks

Use this method after using

[IModelDoc2::PrintPreview](SOLIDWORKS.Interop.sldworks~SOLIDWORKS.Interop.sldworks.IModelDoc2~PrintPreview.html)

to close a Print Preview page.
