# GetModuleCount Method (IMacroFeatureData)

Gets the number of modules for the macro feature.

## Syntax

### Visual Basic (Declaration)

```vb
Function GetModuleCount() As System.Integer
```

### Visual Basic (Usage)

```vb
Dim instance As IMacroFeatureData
Dim value As System.Integer

value = instance.GetModuleCount()
```

### C#

```csharp
System.int GetModuleCount()
```

### C++/CLI

```cpp
System.int GetModuleCount();
```

NOTE:

See

[Differences Between Unmanaged C++ and C++/CLI Code](DifferencesBetweenUnManagedAndCPPCLI.htm)

.

### Return Value

Number of modules

## VBA Syntax

See

[MacroFeatureData::GetModuleCount](ms-its:sldworksapivb6.chm::/sldworks~MacroFeatureData~GetModuleCount.html)

.

## Remarks

Call this method before calling

[IMacroFeatureData::IGetModuleNames](SOLIDWORKS.Interop.sldworks~SOLIDWORKS.Interop.sldworks.IMacroFeatureData~IGetModuleNames.html)

to determine the size of the array for that method.
