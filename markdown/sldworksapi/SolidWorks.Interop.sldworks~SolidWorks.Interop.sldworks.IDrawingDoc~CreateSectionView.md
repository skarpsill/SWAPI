# CreateSectionView Method (IDrawingDoc)

Creates a section view in the drawing using the selected section line.

## Syntax

### Visual Basic (Declaration)

```vb
Sub CreateSectionView()
```

### Visual Basic (Usage)

```vb
Dim instance As IDrawingDoc

instance.CreateSectionView()
```

### C#

```csharp
void CreateSectionView()
```

### C++/CLI

```cpp
void CreateSectionView();
```

[Differences Between Unmanaged C++ and C++/CLI Code](DifferencesBetweenUnManagedAndCPPCLI.htm)

.

See

[DrawingDoc::CreateSectionView](ms-its:sldworksapivb6.chm::/sldworks~DrawingDoc~CreateSectionView.html)

## Remarks

This method prompts the end-user for a name for the new section view.
