# LastError Property (IDesignTable)

Gets or sets the last error that occurred in this design table.

## Syntax

### Visual Basic (Declaration)

```vb
Property LastError As System.Integer
```

### Visual Basic (Usage)

```vb
Dim instance As IDesignTable
Dim value As System.Integer

instance.LastError = value

value = instance.LastError
```

### C#

```csharp
System.int LastError {get; set;}
```

### C++/CLI

```cpp
property System.int LastError {
   System.int get();
   void set (    System.int value);
}
```

NOTE:

See

[Differences Between Unmanaged C++ and C++/CLI Code](DifferencesBetweenUnManagedAndCPPCLI.htm)

.

### Property Value

Last error that occured in this design table as defined in

[swDesignTableErrors_e](ms-its:swconst.chm::/SOLIDWORKS.Interop.swconst~SOLIDWORKS.Interop.swconst.swDesignTableErrors_e.html)

## VBA Syntax

See

[DesignTable::LastError](ms-its:sldworksapivb6.chm::/sldworks~DesignTable~LastError.html)

.

## Remarks

Getting this property returns the last error generated by a request to modify the design table. Setting this property allows you to clear the last design table error that occurred.
