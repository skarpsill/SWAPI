# DestFolderID Property (IEdmBatchItemGeneration)

Gets or sets the ID of the item folder where new items are placed.

## Syntax

```vb

```

### C++/CLI

```cpp

   System.int get();
   void set (    System.int value);
}
```

### Property Value

ID of the item folder; by default, the item root folder
