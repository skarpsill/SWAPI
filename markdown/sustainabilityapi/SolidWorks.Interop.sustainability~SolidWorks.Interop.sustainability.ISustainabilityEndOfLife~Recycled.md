# Recycled Property (ISustainabilityEndOfLife)

Gets or sets the percentage of product that is recycled.

## Syntax

### Visual Basic (Declaration)

```vb
Property Recycled As System.Double
```

### Visual Basic (Usage)

```vb
Dim instance As ISustainabilityEndOfLife
Dim value As System.Double

instance.Recycled = value

value = instance.Recycled
```

### C#

```csharp
System.double Recycled {get; set;}
```

### C++/CLI

```cpp
property System.double Recycled {
   System.double get();
   void set (    System.double value);
}
```

### Property Value

Percentage of the product that is recycled
