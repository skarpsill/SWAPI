# IsSuppressed Method (IDimXpertAnnotation)

Gets whether the annotation is suppressed from view on the DimXpertManager tab of the Management Panel.

## Syntax

### Visual Basic (Declaration)

```vb
Function IsSuppressed() As System.Boolean
```

### Visual Basic (Usage)

```vb
Dim instance As IDimXpertAnnotation
Dim value As System.Boolean

value = instance.IsSuppressed()
```

### C#

```csharp
System.bool IsSuppressed()
```

### C++/CLI

```cpp
System.bool IsSuppressed();
```

### Return Value

True if the annotation is suppressed; false otherwise
