# simple

### Simple code examples

Creating a simple example that models the problem you have in your larger
application makes it easier to communicate the details of a problem or
solution and focus on exactly what is wrong.

This typically means that you should send the minimum code to reproduce
the problem. Almost all problems can be reproduced with a simple VBA macro.
It is rare that a VC++ DLL is needed. Additionally, VBA has the advantage
of rapid development and built in management of COM references.
