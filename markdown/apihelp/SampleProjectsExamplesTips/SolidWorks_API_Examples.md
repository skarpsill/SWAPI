# SOLIDWORKS API Examples, Sample Projects, and Tips

To locate SOLIDWORKS examples in the Help,
use the index tab.

Examples are grouped by language on the Index
tab. Type:

- C#
  examplesto see available C# examples.
- C++/CLI
  examplesto see available managed C++ examples.
- Visual
  Basic .NET (VB.NET) examplesto see available VB.NET examples.
- VBA
  examplesto see available Visual Basic for Applications (VBA) examples.
- C++
  examplesto see available unmanaged C++ examples.

Examples are also linked to relevant reference
topics. See theExamplesection
in a reference topic.

TheSOLIDWORKS
APISupportweb page, which
is part of the[SOLIDWORKS website](http://www.solidworks.com), provides you
access to additional API examples, sample projects, tips, discussion forum,
knowledge base, etc.

NOTE:You
must be connected to the internet to view theSOLIDWORKS
API Supportweb page and SOLIDWORKS website.
