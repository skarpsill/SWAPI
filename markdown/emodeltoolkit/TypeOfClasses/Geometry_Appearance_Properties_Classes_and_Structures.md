# Geometry Appearance Properties Classes and Structures

See Also

(Table)=========================================================

| Class or Structure | Description |
| --- | --- |
| eDAppearance class | Contains color and transparency data and an optional eTexture object. |
| eDColorFloat class | Contains a value between 0.0 and 1.0. |
| eDColorRBG class | Contains three values between 0.0 and 1.0 that represent color data. |
| eDImageByRef class | Contains the dimensions of a bitmap and the bitmap data. |
| eDLineStyle structure | Represents line styles. |
| eDMarkerStyle structure | Represents 3D marker point styles. |
| eDTexture class | Contains bitmap coordinate mapping information and is contained by an eDAppearance . |
