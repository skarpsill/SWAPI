# IsSuppressed Method (ISwDMDimXpertAnnotation)

Gets whether this DimXpert annotation is suppressed from view in the model.

## Syntax

### Visual Basic (Declaration)

```vb
Function IsSuppressed() As System.Boolean
```

### Visual Basic (Usage)

```vb
Dim instance As ISwDMDimXpertAnnotation
Dim value As System.Boolean

value = instance.IsSuppressed()
```

### C#

```csharp
System.bool IsSuppressed()
```

### C++/CLI

```cpp

```

### Return Value

True if suppressed from view; false otherwise
