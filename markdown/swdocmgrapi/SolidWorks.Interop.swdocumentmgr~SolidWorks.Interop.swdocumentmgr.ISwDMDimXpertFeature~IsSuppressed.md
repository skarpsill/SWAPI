# IsSuppressed Method (ISwDMDimXpertFeature)

Gets whether this DimXpert feature is suppressed from the view in the model.

## Syntax

### Visual Basic (Declaration)

```vb
Function IsSuppressed() As System.Boolean
```

### Visual Basic (Usage)

```vb
Dim instance As ISwDMDimXpertFeature
Dim value As System.Boolean

value = instance.IsSuppressed()
```

### C#

```csharp
System.bool IsSuppressed()
```

### C++/CLI

```cpp
System.bool IsSuppressed();
```

### Return Value

True if the feature is suppressed from view; false if not
