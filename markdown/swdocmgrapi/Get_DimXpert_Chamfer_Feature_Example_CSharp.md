# Get DimXpert Chamfer Feature Example (C#)

This example builds a part that demonstrates usage of the SOLIDWORKS Document
Manager API for a DimXpert chamfer feature.

//---------------------------------------------------------------------------- // 1.
Open public_documents \samples\tutorial\api\plate_tolstatus.sldprt. // 2.
Open the DimXpert toolbar from View > Toolbars // (select the first instance
of Toolbars on the View menu). // 3.
Click Auto Dimension Scheme on the DimXpert toolbar. // 4.
Ensure that the chamfer feature filter is selected. // 5.
Click the green check mark to accept all settings. // 6.
Observe the following DimXpert features on // the DimXpertManager tab: Chamfer1 . // 7.
Save the part to another file name. // NOTE: Because
this part is used in a SOLIDWORKS online // tutorial, do not save changes
to the original file name. // 8.
Close the part. // 9.
Read the instructions for loading and running Code_Example_CSharp //
with this part.//-----------------------------------------------------------------------------
