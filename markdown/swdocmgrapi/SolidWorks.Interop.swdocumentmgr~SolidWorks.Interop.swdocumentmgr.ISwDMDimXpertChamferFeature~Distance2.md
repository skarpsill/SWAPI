# Distance2 Property (ISwDMDimXpertChamferFeature)

Gets the length on bevel side 2 of the selected edge, face, or vertex of this DimXpert chamfer.

### Visual Basic (Declaration)

```vb
ReadOnly Property Distance2 As System.Double
```

### Visual Basic (Usage)

```vb

```

### C#

```csharp

```

### C++/CLI

```cpp

   System.double get();
}
```

### Property Value

Length on bevel side 2 of the chamfer
