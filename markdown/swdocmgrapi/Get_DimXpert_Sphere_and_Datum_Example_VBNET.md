# Get DimXpert Sphere and Datum Example (VB.NET)

This example builds a part to demonstrate usage of the SOLIDWORKS Document
Manager API for the following DimXpert entities:

* Sphere feature

* Datum annotation

```vb
 '----------------------------------------------------------------------------
' kadov_tag{{<spaces>}}kadov_tag{{</spaces>}}1. Open public_documents\samples\tutorial\floxpress\ball valve\Ball.sldprt.
' kadov_tag{{<spaces>}}kadov_tag{{</spaces>}}2. Open the DimXpert toolbar from View > Toolbars (select the first
'     instance of Toolbars on the View menu).
' kadov_tag{{<spaces>}}3. Click Datum on the DimXpert toolbar.
' kadov_tag{{<spaces>}}4. Click the ball of the part.
' kadov_tag{{<spaces>}}kadov_tag{{</spaces>}}5. Click outside the part to place the datum annotation.
' kadov_tag{{<spaces>}}kadov_tag{{</spaces>}}6. Click the green check mark to accept the new datum feature.
' kadov_tag{{<spaces>}}kadov_tag{{</spaces>}}7. Observe the following DimXpert feature on the DimXpertManager
'    tab: kadov_tag{{<spaces>}} kadov_tag{{</spaces>}}Sphere1.
' kadov_tag{{<spaces>}}kadov_tag{{</spaces>}}8. Close the part, saving it to another name.
' kadov_tag{{<spaces>}}   NOTE: kadov_tag{{<spaces>}} kadov_tag{{</spaces>}}Because this part is used in a SOLIDWORKS online tutorial,
'           do not save any changes when you close the original file name.
' kadov_tag{{<spaces>}}9. Read how to load and run Code_Example_VBNET with this part.
 '----------------------------------------------------------------------------
```
