# ISwDMConfiguration12 Interface

Accesses configuration-specific information for the part, assembly, or drawing document.

## Remarks

- You can access an ISwDMConfiguration12 object by calling QueryInterface on an[ISwDMConfiguration](SOLIDWORKS.Interop.swdocumentmgr~SOLIDWORKS.Interop.swdocumentmgr.ISwDMConfiguration.html)object in C++ or by direct assignment in Visual Basic.
- The ISwDMConfiguration object can be assigned to an ISwDMConfiguration12 variable, just like the SOLIDWORKS[IModelDoc2](ms-its:sldworksapi.chm::/SOLIDWORKS.Interop.sldworks~SOLIDWORKS.Interop.sldworks.IModelDoc2.html)object can be assigned to a SOLIDWORKS[IPartDoc](ms-its:sldworksapi.chm::/SOLIDWORKS.Interop.sldworks~SOLIDWORKS.Interop.sldworks.IPartDoc.html)variable.
