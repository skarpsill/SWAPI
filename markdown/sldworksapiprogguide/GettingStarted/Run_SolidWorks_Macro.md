# Run SOLIDWORKS Macro

Opens a dialog that lets you select the macro that you want to run.

#### To run a macro:

1. [SOLIDWORKS 2018 offers Visual
  Studio Tools for Applications (VSTA)](VSTA_2015.htm)

  .
2. Click

  Run Macro

  on the Macro toolbar, or click

  Tools
  >

  Macro >

  Run

  .
3. In the dialog, locate the macro (.swpor.dll) file
  that you want to run and clickOpen.

You can also assign SOLIDWORKSmacros
to toolbar buttons.
