# Units

Unless otherwise stated, all SOLIDWORKS API functions use metric units.
Specifically, the SOLIDWORKS API accepts or returns values as meters,
radians, kilograms, square meters, or cubic meters.

**NOTE:**1 radian = 180º/ p = 57.295779513º or approximately 57.3º, which is the conversion value used in
the examples.
