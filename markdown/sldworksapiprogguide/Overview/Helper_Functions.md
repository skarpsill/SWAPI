# Helper Functions

The general helper functions, whose header files were previously provided inpublic_documents\appcomm,
have been superseded by the following API interfaces and their associated methods
and properties:

- [IMathPoint](sldworksAPI.chm::/SolidWorks.interop.sldworks~SolidWorks.interop.sldworks.IMathPoint.html)
- [IMathTransform](sldworksAPI.chm::/SolidWorks.interop.sldworks~SolidWorks.interop.sldworks.IMathTransform.html)
- [IMathUtility](sldworksAPI.chm::/SolidWorks.interop.sldworks~SolidWorks.interop.sldworks.IMathUtility.html)
- [IMathVector](sldworksAPI.chm::/SolidWorks.interop.sldworks~SolidWorks.interop.sldworks.IMathVector.html)
- [IUserUnit](sldworksAPI.chm::/SolidWorks.interop.sldworks~SolidWorks.interop.sldworks.IUserUnit.html)

The old helper functions, which were independent of the SOLIDWORKS API
and provided for your convenience, have been phased out and might not
work in SOLIDWORKS 2001Plus and later.
