# Line Weights

In SOLIDWORKS 2009 and later, SOLIDWORKS recommends that you set document-level
line weights instead of system-level line weights. However, if you must
set system-level line weights on a document created in SOLIDWORKS 2008
or earlier, then you must set the system-level line weights before opening
the document.
