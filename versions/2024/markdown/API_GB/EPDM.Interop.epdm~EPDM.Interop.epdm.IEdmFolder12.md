# IEdmFolder12 Interface

Allows you to access the contents of a file system folder in the vault.

## Remarks

This interface extends

[IEdmFolder11](EPDM.Interop.epdm~EPDM.Interop.epdm.IEdmFolder11.html)

by providing the ability to specify that a new file's data card be created with a name that is either the new file's serial number or the new file's name.
