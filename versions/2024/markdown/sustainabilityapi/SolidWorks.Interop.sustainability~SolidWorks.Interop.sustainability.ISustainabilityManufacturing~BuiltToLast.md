# BuiltToLast Property (ISustainabilityManufacturing)

Gets or sets the length of time that the part is built to last.

## Syntax

### Visual Basic (Declaration)

```vb
Property BuiltToLast As System.Double
```

### Visual Basic (Usage)

```vb
Dim instance As ISustainabilityManufacturing
Dim value As System.Double

instance.BuiltToLast = value

value = instance.BuiltToLast
```

### C#

```csharp
System.double BuiltToLast {get; set;}
```

### C++/CLI

```cpp
property System.double BuiltToLast {
   System.double get();
   void set (    System.double value);
}
```

### Property Value

Length of time that the part is built to last
