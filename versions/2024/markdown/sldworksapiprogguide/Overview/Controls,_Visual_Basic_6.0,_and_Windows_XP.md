# Controls, Visual Basic for Applications (VBA), and Windows XP

If you developed your application using Visual Basic for Applications
(VBA) on Windows XP, then the controls on your forms and dialogs might
behave unexpectedly. Therefore, it is recommended that you port your application
to Microsoft Visual Basic .NET or peruse Microsoft's Knowledge Base and
the internet for possible workarounds.
