# Add Root-level Part or Assembly to Configuration

You cannot add more than one root-level[eDPart](../eDPart/eDPart_Class.htm)or more than one root-level[eDAssembly](../eDAssembly/eDAssembly_Class.htm)to an[eDConfiguration](../eDConfiguration/eDConfiguration_Class.htm)(e.g., you cannot call[eDConfiguration::CreatePart](../eDConfiguration/eDConfiguration_Class.htm)or[eDConfiguration::CreateAssembly](../eDConfiguration/eDConfiguration_Class.htm)more than once for a given configuration).
