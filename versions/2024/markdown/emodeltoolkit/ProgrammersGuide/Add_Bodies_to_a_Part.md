# Add Bodies to a Part

You can add more than one[eDBody](../eDBody/eDBody_Class.htm)to an[eDPart](../eDPart/eDPart_Class.htm). You can, and, if
appropriate, you should, use the same eDBody in more than one eDPart.

Each eDBody can contain an unlimited number of[eDCurveList](../eDCurveList/eDCurveList_Class.htm),[eDMarkerList](../eDMarkerList/eDMarkerList_Class.htm), and[eDShellList](../eDShellList/eDShellList_Class.htm)objects.
