# GetHarmonicNoOfPoints2 Method (ICWDynamicStudyOptions)

Gets the number of points for each frequency of the harmonic dynamic study.

## Syntax

### Visual Basic (Declaration)

```vb
Function GetHarmonicNoOfPoints2( _
   ByRef NPoints As System.Integer _
) As System.Integer
```

### Visual Basic (Usage)

```vb
Dim instance As ICWDynamicStudyOptions

value = instance.GetHarmonicNoOfPoints2(NPoints)
```

### C#

```csharp
System.int GetHarmonicNoOfPoints2(
   out System.int NPoints
)
```

### C++/CLI

```cpp
System.int GetHarmonicNoOfPoints2(
   [Out] System.int NPoints
)
```

### Parameters

- `NPoints`: Number of points for each frequency

### Return Value

0 indicates success; a non-0 value indicates failure
