# ViewFactor Property (ICWRadiation)

Gets or sets the value of the view factor for this radiation.

## Syntax

### Visual Basic (Declaration)

```vb
Property ViewFactor As System.Double
```

### Visual Basic (Usage)

```vb
Dim instance As ICWRadiation
Dim value As System.Double

instance.ViewFactor = value

value = instance.ViewFactor
```

### C#

```csharp
System.double ViewFactor {get; set;}
```

### C++/CLI

```cpp
property System.double ViewFactor {
   System.double get();
   void set (    System.double value);
}
```

### Property Value

Value of view factor
