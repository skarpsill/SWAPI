# ICostBody Interface Methods

For a list of all members of this type, see[ICostBody members](SolidWorks.Interop.sldcostingapi~SolidWorks.Interop.sldcostingapi.ICostBody_members.html).

## Public Methods

|  | Name | Description |
| --- | --- | --- |
| Method | AssignCustomCost | Sets the total cost (setup and operation) of a custom operation for this Costing body. |
| Method | CreateCostAnalysis | Creates a new Costing analysis for this Costing body using the specified Costing template. |
| Method | GetBodyType | Gets the type of Costing body for the Costing analysis, which indicates which Costing template to use. |
| Method | GetCostAnalysis | Accesses the existing Cost analysis for this Costing body. |
| Method | GetName | Gets the name of the Costing body. |
| Method | GetSWBody | Gets the SOLIDWORKS body that corresponds to this Costing body. |
