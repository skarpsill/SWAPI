# SetGuideTangencyType Method (ILoftFeatureData)

Sets the tangency type of the guide curve (i.e., controls the tangency where the loft meets the guide curves) for this loft feature.

## Syntax

### Visual Basic (Declaration)

```vb
Sub SetGuideTangencyType( _
   ByVal GuideIndex As System.Short, _
   ByVal TangType As System.Short _
)
```

### Visual Basic (Usage)

```vb
Dim instance As ILoftFeatureData
Dim GuideIndex As System.Short
Dim TangType As System.Short

instance.SetGuideTangencyType(GuideIndex, TangType)
```

### C#

```csharp
void SetGuideTangencyType(
   System.short GuideIndex,
   System.short TangType
)
```

### C++/CLI

```cpp
void SetGuideTangencyType(
   System.short GuideIndex,
   System.short TangType
)
```

NOTE:

See

[Differences Between Unmanaged C++ and C++/CLI Code](DifferencesBetweenUnManagedAndCPPCLI.htm)

.

### Parameters

- `GuideIndex`: Index of guide curve
- `TangType`: Tangency type of guide curve as defined in[swTangencyType_e](ms-its:swconst.chm::/SOLIDWORKS.Interop.swconst~SOLIDWORKS.Interop.swconst.swTangencyType_e.html)
