# IFeatureStatistics Interface Members

The following tables list the members exposed by[IFeatureStatistics](SolidWorks.Interop.sldworks~SolidWorks.Interop.sldworks.IFeatureStatistics.html).

## Public Properties

|  | Name | Description |
| --- | --- | --- |
| Property | FeatureCount | Gets the number of features in a part document. |
| Property | FeatureNames | Gets the names of the features in a part document. |
| Property | Features | Gets the features in a part document. |
| Property | FeatureTypes | Gets the types of features in a part document. |
| Property | FeatureUpdatePercentageTimes | Gets the percentages of time it takes to update (rebuild) each feature in a part document relative to the total time it takes to update all features in that part document. |
| Property | FeatureUpdateTimes | Gets the times, in seconds, it takes to update (rebuild) each feature in a part document. |
| Property | PartName | Gets the name of the part document whose feature statistics you want. |
| Property | SolidBodiesCount | Gets the number of solid bodies in a part document. |
| Property | SurfaceBodiesCount | Gets the number of surface bodies in a part document. |
| Property | TotalRebuildTime | Gets the time, in seconds, it takes to rebuild (update) all features in a part document. |
