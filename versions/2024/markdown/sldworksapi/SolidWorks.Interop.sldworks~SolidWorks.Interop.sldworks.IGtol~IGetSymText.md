# IGetSymText Method (IGtol)

Gets the symbol text.

## Syntax

### Visual Basic (Declaration)

```vb
Function IGetSymText( _
   ByVal SymIdx As System.Short _
) As System.String
```

### Visual Basic (Usage)

```vb
Dim instance As IGtol
Dim SymIdx As System.Short
Dim value As System.String

value = instance.IGetSymText(SymIdx)
```

### C#

```csharp
System.string IGetSymText(
   System.short SymIdx
)
```

### C++/CLI

```cpp
System.String^ IGetSymText(
   System.short SymIdx
)
```

NOTE:

See

[Differences Between Unmanaged C++ and C++/CLI Code](DifferencesBetweenUnManagedAndCPPCLI.htm)

.

### Parameters

- `SymIdx`: [Symbol index](ms-its:swconst.chm::/SOLIDWORKS.Interop.swconst~SOLIDWORKS.Interop.swconst.swGtolGeomCharSymbol_e.html)number used by SOLIDWORKS

### Return Value

- in-process, unmanaged C++: Pointer to an array of strings. The size of the array is based on the number of text pieces in this GTol. This number can be determined using TextCount returned by

  [IGtol::IGetSymEdgeCounts](SOLIDWORKS.Interop.sldworks~SOLIDWORKS.Interop.sldworks.IGtol~IGetSymEdgeCounts.html)

  .

VBA, VB.NET, C#, and C++/CLI: Not supported

See[In-process Methods](sldworksapiprogguide.chm::/OVERVIEW/In-process_Methods.htm)for details about this type of method.
