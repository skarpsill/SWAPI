# IsCompoundNote Method (INote)

Obsolete for documents created in SOLIDWORKS 2016 and later

. Gets whether the current note is a compound note.

## Syntax

### Visual Basic (Declaration)

```vb
Function IsCompoundNote() As System.Boolean
```

### Visual Basic (Usage)

```vb
Dim instance As INote
Dim value As System.Boolean

value = instance.IsCompoundNote()
```

### C#

```csharp
System.bool IsCompoundNote()
```

### C++/CLI

```cpp
System.bool IsCompoundNote();
```

NOTE:

See

[Differences Between Unmanaged C++ and C++/CLI Code](DifferencesBetweenUnManagedAndCPPCLI.htm)

.

### Return Value

True if note is compound, false if not
