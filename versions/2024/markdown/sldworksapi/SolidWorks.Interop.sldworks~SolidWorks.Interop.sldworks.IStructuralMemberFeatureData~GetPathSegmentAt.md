## Remarks

The body generated by this feature is not available when the feature is rolled back. Therefore, it is not necessary to call[IStructuralMemberFeatureData::AccessSelections](SOLIDWORKS.Interop.sldworks~SOLIDWORKS.Interop.sldworks.IStructuralMemberFeatureData~AccessSelections.html)or[IStructuralMemberFeatureData::ReleaseSelectionAccess](SOLIDWORKS.Interop.sldworks~SOLIDWORKS.Interop.sldworks.IStructuralMemberFeatureData~ReleaseSelectionAccess.html)before or after calling this method.
