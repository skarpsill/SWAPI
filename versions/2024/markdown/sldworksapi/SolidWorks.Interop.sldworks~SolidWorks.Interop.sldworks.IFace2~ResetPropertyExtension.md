# ResetPropertyExtension Method (IFace2)

Clears all values stored in the property extension.

## Syntax

### Visual Basic (Declaration)

```vb
Sub ResetPropertyExtension()
```

### Visual Basic (Usage)

```vb
Dim instance As IFace2

instance.ResetPropertyExtension()
```

### C#

```

```

## Remarks

This method is unsupported for faces obtained from reference surface bodies.
