# ISwQuickTip Interface Properties

For a list of all members of this type, see[ISwQuickTip members](SolidWorks.Interop.swpublished~SolidWorks.Interop.swpublished.ISwQuickTip_members.html).

## Public Properties

|  | Name | Description |
| --- | --- | --- |
| Property | ApplicationName | Gets the name of add-in application. |
| Property | HideGuide | Gets the state of the add-in's Quick Tips. |
| Property | MenuName | Gets the name of add-in's Quick Tips on the SOLIDWORKS Help menu. |
| Property | ShowInAssemblyDoc | Gets whether to show an add-in's Quick Tips in SOLIDWORKS assembly documents. |
| Property | ShowInDrawingDoc | Gets whether to show an add-in's Quick Tips in SOLIDWORKS drawing documents. |
| Property | ShowInPartDoc | Gets whether to show the add-in's Quick Tips in SOLIDWORKS part documents. |
| Property | Url | Gets the URL of the HMTL page to display in the add-in's Quick Tips. |
