# CreateAVIParameter Method (IMotionStudyManager)

Creates the object to use to set and save the AVI parameters of animation videos.

```vb

```

### Visual Basic (Usage)

```vb
value = instance.CreateAVIParameter()
```

### C#

```csharp
AVIParameter CreateAVIParameter()
```

### C++/CLI

```cpp
AVIParameter^ CreateAVIParameter();
```

### Return Value
