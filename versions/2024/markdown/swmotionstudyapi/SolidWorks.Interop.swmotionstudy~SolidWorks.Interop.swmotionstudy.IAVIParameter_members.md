# IAVIParameter Interface Members

The following tables list the members exposed by[IAVIParameter](SolidWorks.Interop.swmotionstudy~SolidWorks.Interop.swmotionstudy.IAVIParameter.html).

## Public Properties

|  | Name | Description |
| --- | --- | --- |
| Property | AspectRatio | Gets or sets the ratio between the width of the image and the height of the image for the animation. |
| Property | BlurLength | Gets or sets the length of the motion blur effect when rendering animations using a ray-trace rendering engine. |
| Property | BlurOffset | Gets or sets the position in time from where the ray-trace rendering engine samples its motion blur for the animation. |
| Property | EndTime | Gets or sets the end time of a specific time period of the animation to save. |
| Property | FramePerSecond | Gets or sets the frames per second in which to play the animation. |
| Property | ImageSize | Gets or sets the image size of the animation. |
| Property | MotionBlur | Gets or sets whether to enable motion blur when saving an animation rendered using a ray-trace rendering engine. |
| Property | OutputType | Gets or sets the type of files to which to save the animation. |
| Property | PreserveRatio | Gets or sets whether to preserve the aspect ratio of the animation when changing either its width or height . |
| Property | RendererType | Gets or sets the renderer type to save the animation. |
| Property | SaveEntireAnimation | Gets or sets whether to save the entire animation. |
| Property | ScreenHeight | Gets or sets the height at which to display the animation on the screen. |
| Property | ScreenRatio | Gets or sets whether to base the aspect ratio on the screen. |
| Property | ScreenWidth | Gets or sets the width at which to display the animation on the screen. |
| Property | StartTime | Gets or sets the start time of a specific time period of the animation to save. |
