# WizardHoleFeatureData::Standard

This
property is obsolete and has been superseded by WizardHoleFeatureData2::Standard .

Description

This property gets or sets the design standard
for this hole.

Syntax (OLE Automation)

standard = WizardHoleFeatureData.Standard ( ) (VB
Get property)

WizardHoleFeatureData.Standard = standard (VB
Set property)

standard = WizardHoleFeatureData.GetStandard ( ) (C++
Get property)

WizardHoleFeatureData.SetStandard =( standard ) (C++
Set property)

(Table)=========================================================

| Return: | (BSTR) standard | Value of the design standard associated with
this hole |
| --- | --- | --- |

Syntax (COM)

status = WizardHoleFeatureData->get_Standard (
&standard ) (COM Get property)

status = WizardHoleFeatureData->put_Standard (
standard ) (COM Set property)

(Table)=========================================================

| Output: | (BSTR) standard | Value of the design standard associated with
this hole |
| --- | --- | --- |
| Return: | (HRESULT) status | S_OK if successful |
