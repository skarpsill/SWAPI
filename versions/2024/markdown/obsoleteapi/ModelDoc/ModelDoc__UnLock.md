# ModelDoc::UnLock

This method is obsolete
and has been superseded by[ModelDoc2::UnLock](sldworksAPI.chm::/ModelDoc2/ModelDoc2__UnLock.htm).

Description

This method removes the effect of ModelDoc::Lock and changes the status
bar message to readProcess Complete.
This method should be used with ModelDoc::Lock.

Syntax (OLE Automation)

void ModelDoc.UnLock ()

Syntax (COM)

status = ModelDoc->UnLock ( )
