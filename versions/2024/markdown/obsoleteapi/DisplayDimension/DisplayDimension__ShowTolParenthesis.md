# DisplayDimension::ShowTolParenthesis

This property is obsolete and has been superseded
by[DimensionTolerance::ShowParenthesis](sldworksAPI.chm::/DimensionTolerance/DimensionTolerance__ShowParenthesis.htm).

Description

This property gets or
sets whether the dimension tolerance is displayed with parenthesis around
the text.

Syntax (OLE Automation)

VB_GET

retval = DisplayDimension.ShowTolParenthesis
(VB Get property)

END_VB_GET

VB_SET

DisplayDimension.ShowTolParenthesis
= show (VB Set property)

END_VB_SET

VC_GET

retval = DisplayDimension.GetShowTolParenthesis (
) (C++ Get property)

END_VC_GET

VC_SET

DisplayDimension.SetShowTolParenthesis
( show ) (C++ Set property)

END_VC_SET

(Table)=========================================================

| Output: | (VARIANT_BOOL) retval | TRUE displays dimension tolerance
text in parenthesis, FALSE does not |
| --- | --- | --- |

COM_GET

END_COM_GET

END_COM_SET
