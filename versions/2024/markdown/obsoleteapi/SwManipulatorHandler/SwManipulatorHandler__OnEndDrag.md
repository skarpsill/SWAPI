# SwManipulatorHandler::OnEndDrag

This method is obsolete and has been superseded
by[SwManipulatorHandler2::OnEndDrag](sldworksAPI.chm::/SwManipulatorHandler2/SwManipulatorHandler2__OnEndDrag.htm).

Description

This method is called when
a user releases a mouse button after dragging the pointer.

Syntax (OLE Automation)

| Input: | (LPDISPATCH) pManipulator | Dispatch pointer to the Manipulator
