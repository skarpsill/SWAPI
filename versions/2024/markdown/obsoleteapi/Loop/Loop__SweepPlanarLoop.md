# Loop::SweepPlanarLoop

This method is obsolete
and has been superseded by[Loop2::SweepPlanarLoop](sldworksAPI.chm::/Loop2/Loop2__SweepPlanarLoop.htm).

Description

This method creates a temporary body object by sweeping a planar loop
along a vector.

Syntax (OLE Automation)

retval = Loop.SweepPlanarLoop ( x,
y, z, draftAngle)

(Table)=========================================================

| Input: | (double) x | X component of the sweep vector |
| --- | --- | --- |
| Input: | (double) y | Y component of the sweep vector |
| Input: | (double) z | Z component of the sweep vector |
| Input: | (double) draftAngle | Desired draft angle for the faces on the side of this swept body |
| Return: | (VARIANT) retval | VARIANT of type SafeArray of 3 objects; the first item in the array
is a Dispatch pointer representing the swept Body object generated by
this method and the next two items are Dispatch pointers to the two stop
faces |

Syntax (COM)

status = Loop->ISweepPlanarLoop
( x, y, z, draftAngle, &stopFacesOut, &retval )

(Table)=========================================================

| Input: | (double) x | X component of the sweep vector |
| --- | --- | --- |
| Input: | (double) y | Y component of the sweep vector |
| Input: | (double) z | Z component of the sweep vector |
| Input: | (double) draftAngle | Desired draft angle for the faces on the side of this swept body |
| Output: | (LPFACE*) stopFacesOut | Pointer to an array of two Face objects that are the two stop faces |
| Output: | (LPBODY) retval | Pointer to the Body object generated by this operation |
| Return: | (HRESULT)status | S_OK if successful |
