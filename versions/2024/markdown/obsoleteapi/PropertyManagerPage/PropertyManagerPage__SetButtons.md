# PropertyManagerPage::SetButtons

This
object and its methods are obsolete and have been superseded by[PropertyManagerPage2](sldworksAPI.chm::/PropertyManagerPage2/PropertyManagerPage2.htm).

Description

This method adds a generic
button to top of the PropertyManager page.

Syntax (OLE Automation)

Not Available.

Syntax (COM)

status = PropertyManagerPage->SetButtons ( ButtonTypes,
&retval )

(Table)=========================================================

| Input: | (long) ButtonTypes | Button types to add to the top of the PropertyManager
page as defined in swPropertyManagerButtonTypes_e |
| --- | --- | --- |
| Output: | (long) retval | Error values as defined in swPropertyManagerStatus_e |
| Return: | (HRESULT) status | S_OK if successful |
