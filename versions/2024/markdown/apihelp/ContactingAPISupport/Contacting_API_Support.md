It is our goal to provide the best support possible for the APIs. Understanding
how this support works, and the type of information we need to assist
you, can help us solve your problems as quickly and efficiently as possible.

#### API Support

An API is a thin layer over the user interface. Thus, if there is a
problem in the API, it often exists in the user interface. If you can
reproduce the problem in the user interface, report the problem to
the user-interface documentation team. Feedback links are at the top of
user-interface documentation pages.

When the problem is fixed in the user interface, then the fix is automatically
applied to the API.

If the problem is only in the API, then you need to create a service request
in the 3D Support Application. Please read the relevant introduction page:

For Customers: https://www.3ds.com/support/discover-3dsupport-app/discover-3dsupport-app-for-customers/

For VARs:

[https://www.3ds.com/support/discover-3dsupport-app/discover-3dsupport-app-for-partners/](https://www.3ds.com/support/discover-3dsupport-app/discover-3dsupport-app-for-partners/)
