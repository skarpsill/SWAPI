# IsTangentPlane Method (ISwDMDimXpertOrientationGeoTol)

Gets whether the tangent plane is specified for this DimXpert orientation geometric tolerance.

## Syntax

```vb

```

```vb
Dim instance As ISwDMDimXpertOrientationGeoTol

```

### C#

```csharp
System.bool IsTangentPlane()
```

### C++/CLI

```cpp

```

### Return Value

True if tangent plane is specified; false if not
