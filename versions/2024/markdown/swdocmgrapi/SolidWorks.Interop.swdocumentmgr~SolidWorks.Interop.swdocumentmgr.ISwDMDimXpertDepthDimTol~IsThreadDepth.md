# IsThreadDepth Method (ISwDMDimXpertDepthDimTol)

Gets whether this dimension tolerance is for thread depth.

## Syntax

### Visual Basic (Declaration)

```vb
Function IsThreadDepth() As System.Boolean
```

### Visual Basic (Usage)

```vb
Dim instance As ISwDMDimXpertDepthDimTol

```

### C#

```csharp

```

### C++/CLI

```cpp

```

### Return Value

True if the tolerance is for thread depth; false otherwise
