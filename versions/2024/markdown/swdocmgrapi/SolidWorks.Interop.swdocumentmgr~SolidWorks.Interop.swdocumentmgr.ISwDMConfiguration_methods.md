# ISwDMConfiguration Interface Methods

For a list of all members of this type, see[ISwDMConfiguration members](SolidWorks.Interop.swdocumentmgr~SolidWorks.Interop.swdocumentmgr.ISwDMConfiguration_members.html).

## Public Methods

|  | Name | Description |
| --- | --- | --- |
| Method | AddCustomProperty | Adds a custom property to this configuration. |
| Method | DeleteCustomProperty | Deletes the specified custom property from this configuration. |
| Method | GetBodiesCount | Gets the number of solid bodies for this configuration. |
| Method | GetBody | Gets the body at the specified index in the specified Parasolid binary file (filename extension .x_b ). |
| Method | GetCustomProperty | Gets the specified custom property for this configuration. |
| Method | GetCustomPropertyCount | Gets the number of custom properties for this configuration. |
| Method | GetCustomPropertyNames | Gets the names of the custom properties for this configuration. |
| Method | GetLastUpdateStamp | Gets the date that this configuration was last updated. |
| Method | GetMassProperties | Gets the mass properties, if valid, for this configuration. |
| Method | GetParentConfigurationName | Gets the name of the parent configuration for this configuration. |
| Method | GetPreviewBitmap | Gets the preview bitmap for this configuration. |
| Method | SetCustomProperty | Obsolete. Superseded by ISwDMConfiguration18::SetCustomProperty2 . |
