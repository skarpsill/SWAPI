# ISwDMConfiguration5 Interface

Accesses configuration-specific information for the part, assembly, or drawing document.

NOTE:

Click the

Members

link, located near the top of the topic, to see this interface's methods and properties.

## Syntax

### Visual Basic (Declaration)

```vb
Public Interface ISwDMConfiguration5
   Inherits ISwDMConfiguration, ISwDMConfiguration2, ISwDMConfiguration3, ISwDMConfiguration4
```

### Visual Basic (Usage)

```vb
Dim instance As ISwDMConfiguration5
```

### C#

```csharp
public interface ISwDMConfiguration5 : ISwDMConfiguration, ISwDMConfiguration2, ISwDMConfiguration3, ISwDMConfiguration4
```

### C++/CLI

```cpp
public interface class ISwDMConfiguration5 : public ISwDMConfiguration, ISwDMConfiguration2, ISwDMConfiguration3, ISwDMConfiguration4
```

## Remarks

- You can access an ISwDMConfiguration5 object by calling QueryInterface on an[ISwDMConfiguration](SOLIDWORKS.Interop.swdocumentmgr~SOLIDWORKS.Interop.swdocumentmgr.ISwDMConfiguration.html)object in C++ or by direct assignment in Visual Basic.
- The ISwDMConfiguration object can be assigned to an ISwDMConfiguration5 variable, just like the SOLIDWORKS[IModelDoc2](ms-its:sldworksapi.chm::/SOLIDWORKS.Interop.sldworks~SOLIDWORKS.Interop.sldworks.IModelDoc2.html)object can be assigned to a SOLIDWORKS[IPartDoc](ms-its:sldworksapi.chm::/SOLIDWORKS.Interop.sldworks~SOLIDWORKS.Interop.sldworks.IPartDoc.html)variable.
