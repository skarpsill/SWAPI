# ModuleName Property (IMacroFeatureData)

Gets or sets the name of a module in the macro for this macro feature.

## Syntax

### Visual Basic (Declaration)

```vb
Property ModuleName As System.String
```

### Visual Basic (Usage)

```vb
Dim instance As IMacroFeatureData
Dim value As System.String

instance.ModuleName = value

value = instance.ModuleName
```

### C#

```csharp
System.string ModuleName {get; set;}
```

### C++/CLI

```cpp
property System.String^ ModuleName {
   System.String^ get();
   void set (    System.String^ value);
}
```

NOTE:

See

[Differences Between Unmanaged C++ and C++/CLI Code](DifferencesBetweenUnManagedAndCPPCLI.htm)

.

### Property Value

Name of the module in the macro
