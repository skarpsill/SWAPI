# ITexture Interface Properties

For a list of all members of this type, see[ITexture members](SolidWorks.Interop.sldworks~SolidWorks.Interop.sldworks.ITexture_members.html).

## Public Properties

|  | Name | Description |
| --- | --- | --- |
| Property | Angle | Gets or sets the rotation angle of the texture. |
| Property | BlendColor | Gets or sets whether to blend the part color with the texture color. |
| Property | DecalType | Gets whether the texture is a decal. |
| Property | IgnoreColorMaterial | Gets whether to ignore the color texture-based appearance. |
| Property | MaterialName | Gets or sets the path and file name of the texture material. |
| Property | ScaleFactor | Gets or sets the granularity of the texture. |
| Property | TransX | Gets the x coordinate for the translation point of the texture-based appearance. |
| Property | TransY | Gets the y coordinate for the translation point of the texture-based appearance. |
| Property | UDir | Gets the U value of the texture-based appearance. |
| Property | VDir | Gets the V value of the texture-based appearance. |
| Property | VScale | Gets the scale for this texture-based appearance or decal. |
