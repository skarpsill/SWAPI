# GetProcedureNames Method (IMacroFeatureData)

Gets the names of the procedures in the specified module for the macro for the macro feature.

## Syntax

### Visual Basic (Declaration)

```vb
Function GetProcedureNames( _
   ByVal ModuleName As System.String _
) As System.Object
```

### Visual Basic (Usage)

```vb
Dim instance As IMacroFeatureData
Dim ModuleName As System.String
Dim value As System.Object

value = instance.GetProcedureNames(ModuleName)
```

### C#

```csharp
System.object GetProcedureNames(
   System.string ModuleName
)
```

### C++/CLI

```cpp
System.Object^ GetProcedureNames(
   System.String^ ModuleName
)
```

NOTE:

See

[Differences Between Unmanaged C++ and C++/CLI Code](DifferencesBetweenUnManagedAndCPPCLI.htm)

.

### Parameters

- `ModuleName`: Name of module

### Return Value

Array of procedure names in module ModuleName
