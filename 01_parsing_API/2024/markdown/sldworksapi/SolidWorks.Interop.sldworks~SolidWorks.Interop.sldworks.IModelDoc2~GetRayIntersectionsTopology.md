# GetRayIntersectionsTopology Method (IModelDoc2)

[IModelDoc2::RayIntersections](SOLIDWORKS.Interop.sldworks~SOLIDWORKS.Interop.sldworks.IModelDoc2~RayIntersections.html)

## Remarks

This method only returns information if the swRayPtsOptsTOPOLS option was specified in the Options argument of[IModelDoc2::RayIntersections](SOLIDWORKS.Interop.sldworks~SOLIDWORKS.Interop.sldworks.IModelDoc2~RayIntersections.html).

The array returned contains information about each entity hit of each ray on the given bodies. The topology hit can be coordinated with the hit point using the BodyIndex return value from[IModelDoc2::GetRayIntersectionsPoints](SOLIDWORKS.Interop.sldworks~SOLIDWORKS.Interop.sldworks.IModelDoc2~IGetRayIntersectionsPoints.html). You can determine the type of item hit (face, edge or vertex) by checking the IntersectionType return value in the IModelDoc2::GetRayIntersectionsPoints.
