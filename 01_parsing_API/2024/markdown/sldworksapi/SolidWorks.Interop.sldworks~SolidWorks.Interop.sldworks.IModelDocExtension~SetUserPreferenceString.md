# SetUserPreferenceString Method (IModelDocExtension)

Sets document default user preference values. This method is intended for user preferences of type string.

## Syntax

### Visual Basic (Declaration)

```vb
Function SetUserPreferenceString( _
   ByVal UserPref As System.Integer, _
   ByVal UserPrefOption As System.Integer, _
   ByVal Value As System.String _
) As System.Boolean
```

### Visual Basic (Usage)

```vb
Dim instance As IModelDocExtension
Dim UserPref As System.Integer
Dim UserPrefOption As System.Integer
Dim Value As System.String
Dim value As System.Boolean

value = instance.SetUserPreferenceString(UserPref, UserPrefOption, Value)
```

### C#

```csharp
System.bool SetUserPreferenceString(
   System.int UserPref,
   System.int UserPrefOption,
   System.string Value
)
```

### C++/CLI

```cpp
System.bool SetUserPreferenceString(
   System.int UserPref,
   System.int UserPrefOption,
   System.String^ Value
)
```

NOTE:

See

[Differences Between Unmanaged C++ and C++/CLI Code](DifferencesBetweenUnManagedAndCPPCLI.htm)

.

### Parameters

- `UserPref`: User preference as defined in

[swUserPreferenceStringValue_e](ms-its:swconst.chm::/SOLIDWORKS.Interop.swconst~SOLIDWORKS.Interop.swconst.swUserPreferenceStringValue_e.html)
- `UserPrefOption`: User-preference option for customizing annotation and dimension drafting standards and as defined in

[swUserPreferenceOption_e](ms-its:swconst.chm::/SOLIDWORKS.Interop.swconst~SOLIDWORKS.Interop.swconst.swUserPreferenceOption_e.html)

; if not needed, then specify swDetailingNoOptionSpecified
- `Value`: Value of the user preference specified in UserPreference

### Return Value

True if user preference is set, false if not
