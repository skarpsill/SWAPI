# Top Property (IPropertyManagerPageControl)

Gets or sets the top edge of the control on a PropertyManager page.

## Syntax

### Visual Basic (Declaration)

```vb
Property Top As System.Short
```

### Visual Basic (Usage)

```vb
Dim instance As IPropertyManagerPageControl
Dim value As System.Short

instance.Top = value

value = instance.Top
```

### C#

```csharp
System.short Top {get; set;}
```

### C++/CLI

```cpp
property System.short Top {
   System.short get();
   void set (    System.short value);
}
```

NOTE:

See

[Differences Between Unmanaged C++ and C++/CLI Code](DifferencesBetweenUnManagedAndCPPCLI.htm)

### Property Value

Top edge of the control
