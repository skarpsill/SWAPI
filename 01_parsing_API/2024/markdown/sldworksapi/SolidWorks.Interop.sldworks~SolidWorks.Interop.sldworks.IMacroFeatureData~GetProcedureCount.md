# GetProcedureCount Method (IMacroFeatureData)

Gets the number of procedures in the specified module in the macro for this macro feature.

## Syntax

### Visual Basic (Declaration)

```vb
Function GetProcedureCount( _
   ByVal ModuleName As System.String _
) As System.Integer
```

### Visual Basic (Usage)

```vb
Dim instance As IMacroFeatureData
Dim ModuleName As System.String
Dim value As System.Integer

value = instance.GetProcedureCount(ModuleName)
```

### C#

```csharp
System.int GetProcedureCount(
   System.string ModuleName
)
```

### C++/CLI

```cpp
System.int GetProcedureCount(
   System.String^ ModuleName
)
```

NOTE:

See

[Differences Between Unmanaged C++ and C++/CLI Code](DifferencesBetweenUnManagedAndCPPCLI.htm)

.

### Parameters

- `ModuleName`: Name of module

### Return Value

Number of procedures in module ModuleName

## VBA Syntax

See

[MacroFeatureData::GetProcedureCount](ms-its:sldworksapivb6.chm::/sldworks~MacroFeatureData~GetProcedureCount.html)

.

## Remarks

Call this method before calling

[IMacroFeatureData::IGetProcedureNames](SOLIDWORKS.Interop.sldworks~SOLIDWORKS.Interop.sldworks.IMacroFeatureData~IGetProcedureNames.html)

to determine the size of the array for that method.
