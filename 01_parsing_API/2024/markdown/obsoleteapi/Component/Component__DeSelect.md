# Component::DeSelect

This
method is obsolete and has been superseded by[Component2::DeSelect](sldworksAPI.chm::/Component2/Component2__DeSelect.htm).

Description

This method deselects this component.

Syntax (OLE Automation)

retval = Component.DeSelect ( )

(Table)=========================================================

| Return: | (BOOL) retval | TRUE if the component was deselected, FALSE if
not |
| --- | --- | --- |

Syntax (COM)

status = Component->DeSelect ( &retval )

(Table)=========================================================

| Output: | (VARIANT_BOOL) retval | TRUE if the component was deselected, FALSE if
not |
| --- | --- | --- |
| Return: | (HRESULT) status | S_OK if successful |
