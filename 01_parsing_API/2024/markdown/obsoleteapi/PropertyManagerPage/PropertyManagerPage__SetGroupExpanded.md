# PropertyManagerPage::SetGroupExpanded

This
object and its methods are obsolete and have been superseded by[PropertyManagerPage2](sldworksAPI.chm::/PropertyManagerPage2/PropertyManagerPage2.htm).

Description

This method sets whether or not this group
box is expanded on this PropertyManager page.

Syntax (OLE Automation)

Not Available.

Syntax (COM)

status = PropertyManagerPage->SetGroupExpanded
( GroupId, Expanded )

(Table)=========================================================

| Input: | (long) GroupId | Resource ID of the group title's static text
control |
| --- | --- | --- |
| Input: | (VARIANT_BOOL) Expanded | TRUE if the group box is to be expanded, otherwise
FALSE |
| Return: | (HRESULT) status | S_OK if successful |
