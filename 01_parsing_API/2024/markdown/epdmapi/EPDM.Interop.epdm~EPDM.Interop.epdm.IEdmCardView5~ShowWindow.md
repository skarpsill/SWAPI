# ShowWindow Method (IEdmCardView5)

Sets whether to show or hide the card view window.

## Syntax

### Visual Basic

```vb
Sub ShowWindow( _
   Optional ByVal bShow As System.Boolean _
)
```

### C#

```csharp
void ShowWindow(
   System.bool bShow
)
```

### C++/CLI

```cpp
void ShowWindow(
   System.bool bShow
)
```

### Parameters

- `bShow`: True to show the card view window, false to not
