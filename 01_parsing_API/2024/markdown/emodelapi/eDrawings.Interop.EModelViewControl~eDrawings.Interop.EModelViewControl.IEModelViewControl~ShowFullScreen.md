# ShowFullScreen Method (IEModelViewControl)

Maximizes the drawing to fill the screen.

## Syntax

### Visual Basic (Declaration)

```vb
Sub ShowFullScreen( _
   ByVal fullscreen As System.Boolean _
)
```

### Visual Basic (Usage)

```vb
Dim instance As IEModelViewControl
Dim fullscreen As System.Boolean

instance.ShowFullScreen(fullscreen)
```

### C#

```csharp
void ShowFullScreen(
   System.bool fullscreen
)
```

### C++/CLI

```cpp
void ShowFullScreen(
   System.bool fullscreen
)
```

### Parameters

- `fullscreen`: True to maximize the drawing; false to not
