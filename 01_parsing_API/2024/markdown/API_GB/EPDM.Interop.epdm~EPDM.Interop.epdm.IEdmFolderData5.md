# IEdmFolderData5 Interface

Allows you to specify user and group permissions and file data cards when creating a folder with[IEdmFolder5::AddFolder](EPDM.Interop.epdm~EPDM.Interop.epdm.IEdmFolder5~AddFolder.html).

## Remarks

This interface inherits from IDispatch. See[IDispatch Interface (Automation)](http://msdn.microsoft.com/en-us/library/windows/desktop/ms221608(v=vs.85).aspx).

By default, user permissions and file data card definitions are inherited from the parent folder.
