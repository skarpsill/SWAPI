## Remarks

After setting this property to 1, specify[ICWThermalStudyOptions::FlowConvectionCoefFile](SolidWorks.Interop.cosworks~SolidWorks.Interop.cosworks.ICWThermalStudyOptions~FlowConvectionCoefFile.html)with the result file generated by SOLIDWORKS Flow Simulation (***.fld**).

This property is not valid for 2D simplification studies.
