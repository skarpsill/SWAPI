# Unmanaged C++ and C++/CLI Code Differences

Many differences exist between unmanaged C++ and .NET-based C++/CLI
(also called managed C++). One difference between unmanaged C++ and C++/CLI
is the syntax. Although this Help shows the syntax for C++/CLI only, SOLIDWORKS
supports both unmanaged C++ and C++/CLI.

To help you transform C++/CLI syntax to unmanaged C++ syntax, examine
the examples in the following table. The same example is written in unmanaged
C++ and C++/CLI code.

(Table)=========================================================

| Example | Code |
| --- | --- |
| Traverse Features | Unmanaged C++ C++/CLI |
| Traverse Configurations | Unmanaged
C++ (demonstrates a BSTR* C-array) Unmanaged
C++ (demonstrates a VARIANT array) C++/CLI
(demonstrates an array) |
| Select Objects | Unmanaged C++ C++/CLI |

To learn more about the differences between unmanaged C++ and C++/CLI,
SOLIDWORKS recommends that you consult the internet and books containing
information about their differences.
