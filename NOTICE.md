files into Markdown and compact LLM navigation indexes.

The `markdown/` directory, when included, is an example generated corpus based
on SOLIDWORKS API 2024 help files. SOLIDWORKS, Dassault Systemes, and related
product names are trademarks or registered trademarks of their respective
owners. The generated Markdown may contain material derived from official
SOLIDWORKS API documentation and may be subject to the original owner's rights
and license terms.

Original `.chm` files and intermediate extracted HTML files are intentionally
not included in this repository. Users should generate documentation from API
help files they are authorized to use.
